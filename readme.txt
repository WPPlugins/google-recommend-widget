Widget that displays a Google recommend (+1) button.

== Description ==

This plugin displays a Google Recommend (+1) button as a widget on your site.

If you have any questions, comments, or suggestions please contact me.

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the Google Recommend Widget to your site through the widget menu.

== Frequently Asked Questions ==

= The widget does not display on mobile devices. How come? =

Googles answer to this is: "We are not including the +1 button on mobile search results at this time.".
